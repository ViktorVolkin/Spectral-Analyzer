# Анализ спектра отражения

Скрипт для анализа спектра отражения: находит два верхних экстремума, нижний экстремум между ними и вычисляет разность ординат.

## Установка зависимостей

Рекомендуется использовать виртуальное окружение:

```bash
# Создание виртуального окружения
python3 -m venv venv

# Активация виртуального окружения
source venv/bin/activate  # для macOS/Linux
# или
venv\Scripts\activate  # для Windows

# Установка зависимостей
pip install -r requirements.txt
```

После активации виртуального окружения используйте `python` и `pip` вместо `python3` и `pip3`.

## Использование

```bash
python3 spectrum_analysis.py <путь_к_csv_файлу> [опции]
```

### Параметры командной строки

- `input_file` - путь к CSV файлу с данными (обязательный)
- `--window` - длина окна для сглаживания (по умолчанию: 51)
- `--polyorder` - порядок полинома для сглаживания (по умолчанию: 3)
- `--output` - имя файла для сохранения графика (опционально)
- `--no-plot` - не показывать графики

### Примеры

```bash
# Базовое использование
python3 spectrum_analysis.py example_spectrum.csv

# С сохранением графика
python3 spectrum_analysis.py example_spectrum.csv --output spectrum_plot.png

# С настройкой параметров сглаживания
python3 spectrum_analysis.py example_spectrum.csv --window 31 --polyorder 2

# Без отображения графиков
python3 spectrum_analysis.py example_spectrum.csv --no-plot
```

## Формат входных данных

CSV файл должен содержать как минимум 2 столбца:
- Первый столбец: абсциссы (например, длина волны)
- Второй столбец: ординаты (например, коэффициент отражения)

Пример:
```csv
wavelength,reflectance
400,0.15
410,0.18
...
```

## Выходные данные

Скрипт выводит:
1. Информацию о загруженных данных
2. Координаты найденных пиков и долины
3. **Результат: разность ординат** (основной ответ)
4. Графики (если не указан флаг `--no-plot`):
   - Исходный спектр с выделенными точками
   - Сглаженный спектр с выделенными точками и прямой между пиками
   - Первая производная
   - Вторая производная

## Алгоритм

1. Загрузка данных из CSV файла
2. Сглаживание спектра методом Савицкого-Голея
3. Поиск локальных экстремумов (пиков и долин)
4. Выбор двух верхних пиков
5. Поиск долины между выбранными пиками
6. Вычисление разности ординат между долиной и точкой на прямой, соединяющей пики
7. Построение графиков и вывод результата
